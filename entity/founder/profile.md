# 创始人工作档案

本手册记录创始人的工作习惯，形成两套档案体系。

## 背景

创始人的工作内容涉及两种性质：
- **可公开内容**：工作方法论、决策思路、项目复盘
- **内部内容**：讨论过程、敏感决策、个人思考

因此建立两套档案系统，分别服务不同的需求。

## 飞书知识库内部档案

- **定位**：内部协作与过程记录
- **存放内容**：
  - 会议讨论与临时想法
  - 敏感业务信息
  - 内部决策过程

## 协同关系

### 内容流动

```
备忘录 → GitHub → 飞书知识库
飞书 → GitHub（可公开） / 飞书知识库（内部）
```

### 选取原则

| 内容类型 | 存放位置 |
|----------|----------|
| 工作方法论 | GitHub |
| 可公开的复盘 | GitHub |
| 团队讨论 | 飞书 |
| 敏感业务 | 飞书 |
| 临时想法 | 优先飞书，可整理后转 GitHub |
