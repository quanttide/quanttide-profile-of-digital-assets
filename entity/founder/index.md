# 资产目录

本项目以记忆系统为隐喻，构建创始人的知识资产体系。

## 记忆模型

```
工作记忆（即时可用）
├── profile（工作档案） - 陈述性记忆
└── handbook（工作手册） - 程序性记忆

长期记忆（沉淀积累）
├── 工作札记 - 陈述性记忆
└── 工作报告 - 程序性记忆
```

## 目录结构

```
asset/
├── profile/    # 工作档案（陈述性记忆）
├── handbook/   # 工作手册（程序性记忆）
└── notes/     # 工作札记
    └── report # 工作报告
```

## 工作档案（profile）

陈述性记忆：可描述、可回忆的事实性知识

- 工作方法论
- 决策思路
- 项目经验

| 位置 | 访问范围 |
|------|----------|
| GitHub | 公开 |
| 飞书知识库 | 内部 |

## 工作手册（handbook）

程序性记忆：可执行、可操作的方法指南

- 两大工作模式（个人 / 协作）
- 工具链规范
- 流程与模板

## 工作札记

长期记忆的沉淀与备份，陈述性记忆

- 微信公众号
- 知乎
- 简书
- 其他平台

**特点**：碎片化、即时性、个人视角

## 工作报告

程序性记忆：系统梳理与总结

- 阶段性总结
- 项目复盘
- 方法论沉淀

**特点**：结构化、系统性、深度分析

## 记忆流转

```
个人模式（工作档案/手册）
备忘录 → GitHub → 飞书知识库

工作札记（独立创作）
直接创作 → 各平台发布

协作模式（工作档案/手册）
飞书 → GitHub / 飞书知识库
```

## 资产关系

| 类型 | 记忆类型 | 特点 | 用途 |
|------|----------|------|------|
| 工作档案 | 陈述性 | 事实性知识 | 知识沉淀 |
| 工作手册 | 程序性 | 操作指南 | 团队学习 |
| 工作札记 | 陈述性 | 碎片思考 | 内容发布 |
| 工作报告 | 程序性 | 系统总结 | 深度复盘 |
